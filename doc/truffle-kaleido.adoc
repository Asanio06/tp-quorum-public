=== Configuration de Truffle pour Kaleido
ifndef::chapter-img[:chapter-img: ./img]

Afin de pouvoir faire des appels depuis Truffle et Web3 vers l'infrastructure
précédemment créée dans Kaleido, il va nous falloir récupérer les identifiants
de connexion pour chaque participant (éleveurs, laiterie, coopérative, etc).

Nous allons commencer par la _Coopérative_ :

.Accès à l'écran de création des identifiants
image::{chapter-img}/kaleido-credentials-1.png[align=center, title-align=center]

.Accès à l'écran de création des identifiants
image::{chapter-img}/kaleido-credentials-2.png[align=center, title-align=center]

.Accès à l'écran de création des identifiants
image::{chapter-img}/kaleido-credentials-3.png[align=center, title-align=center]

CAUTION: N'oubliez surtout pas de cliquer sur le lien *Copy* car c'est la
  dernière fois que Kaleido vous présentera ces informations ! +
  Vu que le compte gratuit est utilisé en totalité, si cette étape est ratée il
  faudra recommencer tout depuis un autre compte !

Vous avez donc maintenant de copié dans le presse-papier de votre ordinateur
l'URL de connexion sécurisé pour le noeud de la _Coopérative_.

Nous allons modifier le fichier ```truffle-config.js``` pour l'enrichir de
réglages de connexion au noeud de la _Coopérative_ :

[source,javascript]
.truffle-config.js
----
include::../truffle-config.js[tags=cooperative]
----
<1> Remplacer cet URL par celui contenu dans le presse-papier.

Répétez ces opérations pour les autres noeuds, à savoir :

* ```Eleveur Hauteluce```
* ```Eleveur Parly```
* ```Eleveur Bastia```
* ```Laiterie Beaufort```

Vous devriez avoir à la fin un fichier ```truffle-config.js``` proche de celui
ci :
